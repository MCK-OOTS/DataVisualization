# 출산율 관련 데이터 분석 및 시각화 프로젝트
---
## 주제선정
- 최근 몇년간 심각한 사회문제 중 하나로 떠오른 대한민국의 출산율 문제.<br>
- 출산율의 원인으로 집 값, 청년실업, 늦은 사회진출 등을 다양한 원인이 제기 되고 있다.<br>
- 이에 정말 그런지 관련데이터를 시각화하고 분석해보기로 했다.<br>
---
## 개발환경 및 라이브러리
- Google Colaboratory(Colab)
- Pandas
- Matplotlib
- Seaborn
